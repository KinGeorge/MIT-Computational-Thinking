# MIT-Computational-Thinking
[Course Website](https://computationalthinking.mit.edu/Fall22/)
## Install Julia 1.8.5(MAC)

```shell
brew install julia
```
check the version:
```shell
julia -v
```
```shell
julia> ]
(@v1.8) pkg> add Pluto
```
start Pluto
```shell
julia> using Pluto
julia> Pluto.run()
```
